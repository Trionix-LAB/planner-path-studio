## Summary
Обновить блок миссий на стартовом экране: переименовать в «Миссии», поддержать сортировку по дате и имени, добавить пагинацию и ограничить вывод 5 элементами на страницу по умолчанию.

## Description (from issue)
- Переименовать список «Недавние миссии» на «Миссии».
- Добавить сортировку по дате и по имени.
- Добавить навигацию по страницам.
- Ограничить отображение до 5 миссий в окне.

## Fixed decisions
- Навигация по списку: pagination.
- Infinite scroll не используется.
- Параметр лимита: 5 записей на страницу по умолчанию (configurable в реализации без отдельной UI-настройки), согласно R-014.

## Acceptance criteria
- [ ] На стартовом экране заголовок блока — «Миссии».
- [ ] Доступны сортировки: по дате (убывание/возрастание) и по имени (алфавитно).
- [ ] Реализована pagination-навигация по страницам.
- [ ] На странице по умолчанию отображается не более 5 миссий.
- [ ] Показываются имя миссии, путь и human-readable дата последнего изменения.
- [ ] При отсутствии миссий отображается placeholder «Нет доступных миссий».
- [ ] Поведение StartScreen и OpenMissionDialog использует общую реализацию/хук.
- [ ] Добавлены/обновлены unit-тесты для загрузки, сортировки, пагинации и пустого состояния.
