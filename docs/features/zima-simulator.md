# test(zima): Zima simulator (UDP NMEA GGA) for local and CI testing

## Цель
Простой симулятор устройства Zima для локального тестирования и использования в CI: UDP генератор NMEA GGA-пакетов с поддержкой корректных и некорректных сценариев.

## Область и режимы
- CLI/Node скрипт (`tools/zima-sim`) с параметрами: порт, интерфейс, rate, режим (single, stream, playback)
- Форматы сценариев: примерный YAML/JSON с последовательностью NMEA-строк и задержками
- Режимы отправки: корректные GGA, случайные/битые строки, частично обрезанные пакеты, проигрывание трасс
- Опционально: Docker image для запуска в CI

## Требования
- Конфигурируемый порт и интерфейс
- Флаги для включения: отправлять только корректные строки / смесь / только битые
- CLI API: `start`, `stop`, `load-scenario`, `status`
- Примеры сценариев и документация в `docs/` или `tools/`
- Интеграция в тесты репозитория (интеграционные тесты, использующие симулятор)

## Критерии приёмки
- [ ] CLI запускается и отправляет NMEA-пакеты на указанный порт
- [ ] Наличие сценариев корректного/некорректного потока
- [ ] Интеграционные тесты, использующие симулятор
- [ ] Документация и пример запуска в README
