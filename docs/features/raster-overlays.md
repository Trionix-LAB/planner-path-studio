# Растровые наложения (GeoTIFF/TFW)

## Что поддерживается в MVP
- Импорт `*.tif`/`*.tiff` с геопривязкой GeoTIFF.
- Импорт пары `*.tif` + `*.tfw`.
- Управление слоями: видимость, прозрачность, порядок (`z-index`), переход к растру.
- Переключатель базовой карты `Тайловая подложка` находится в секции `Слои`.
- Кнопки порядка `↑/↓` в элементах управления растрами перемещают слой ровно на один уровень вверх/вниз (между соседними слоями).

## Ограничения
- CRS: для GeoTIFF только `EPSG:4326`.
- Для `TIF + TFW` поддержаны режимы интерпретации:
  - `градусы` (`EPSG:4326`);
  - `WebMercator` (`EPSG:3857`, метры);
  - `UTM зона` (метры, с выбором зоны `1..60` и полушария `N/S`).
- Для `TFW` поворот (`B/D != 0`) не поддерживается.
- При ошибке геопривязки слой не импортируется, показывается сообщение об ошибке.

## Формат TFW
- `TFW` обрабатывается как world file из ровно 6 строк: `A`, `D`, `B`, `E`, `C`, `F`.
- `A` и `E` задают размер пикселя (обычно `E < 0` для north-up изображений).
- `C` и `F` — координаты центра верхнего левого пикселя.
- Пустая строка допускается только в конце файла; лишние/пропущенные строки считаются ошибкой формата.
- Файл `TFW` должен иметь то же базовое имя, что и `TIF/TIFF` (`karta.tif` + `karta.tfw`).
- Пара `TIF/TFW` должна быть в одной папке (если платформа передает исходные пути файлов).

## Как импортировать
1. Откройте выпадающее меню миссии на верхней панели (`Черновик`/имя миссии).
2. Выберите нужный пункт:
   - Откройте `Импорт` -> `Импорт TIF`.
   - `Встроенная привязка (GeoTIFF)` — только файлы `*.tif/*.tiff` с геопривязкой внутри.
   - `TIF + TFW (градусы)` — выбирайте один или несколько `*.tif/*.tiff`; для каждого файла приложение автоматически ищет одноименный `*.tfw` в той же папке.
   - `TIF + TFW (WebMercator, EPSG:3857)` — выбирайте один или несколько `*.tif/*.tiff`; `*.tfw` ищется автоматически в той же папке.
  - `TIF + TFW (UTM зона)` — открывается отдельное окно импорта UTM, где задаются:
    - номер UTM-зоны (только `1..60`);
    - полушарие (`N`/`S`).
    Затем выбирайте один или несколько `*.tif/*.tiff`; `*.tfw` ищется автоматически в той же папке.
    Если номер зоны вне диапазона `1..60`, кнопка выбора `TIF` блокируется и показывается ошибка валидации.
   - Для `TIF + TFW` встроенная геопривязка в `TIF` не требуется: координаты берутся из автоматически найденного `TFW`.
3. Подтвердите выбор файлов в диалоге.

## Где хранится
- Метаданные: `mission.json` -> `ui.raster_overlays`.
- Данные растра: в папке миссии `overlays/rasters/*`.
- При импорте `GeoTIFF` копируется `TIF/TIFF` в `overlays/rasters/`.
- При импорте `TIF + TFW` копируются оба файла: `TIF/TIFF` и соответствующий `TFW` в `overlays/rasters/`.
- После импорта отображение и дальнейшая работа идут по копиям из папки миссии.
