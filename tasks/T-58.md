# T-58: Arbitrary diver marker size in UI

**Issue:** #58
**Spec:** R-044 (updated)

## Why
* Во вкладке "Агенты" размер маркера ограничен значениями `12..64`, из-за чего нельзя задать произвольный размер для конкретного сценария работы.
* Ограничение зашито в нескольких слоях (UI, нормализация данных, рендер), что приводит к одинаковому багу даже при ручном вводе.

## Goal
* Разрешить ввод произвольного положительного размера маркера (с единым диапазоном валидации).
* Обеспечить единый контракт валидации во всех слоях: форма, нормализация конфигурации, рендер на карте.
* Зафиксировать новое поведение в спецификации и покрыть тестами.
