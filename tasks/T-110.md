# T-110: Инструмент «Измерить» — измерение расстояния между двумя точками

**Issue:** #110
**Branch:** `feat/map-110-measure-tool`
**Spec:** R-064 (новое требование)

---

## Why

Пользователям необходимо измерять расстояния на карте прямо в приложении, без внешних инструментов. Измерение должно быть полноценным объектом миссии: сохраняться, редактироваться (название, цвет, описание) и удаляться.

## Goal

Добавить кнопку «Измерить» в блок «Инструменты» TopBar. При активации: первый клик фиксирует начальную точку, второй — конечную; на карте отрисовывается пунктирная линия с подписью расстояния в метрах. Объект «Измерение» появляется в списке объектов и редактируется в правой панели свойств. ПКМ по кнопке «Измерить» — диалог «Удалить все измерения?».

### Архитектурный паттерн

Приложение использует **prop-drilling** (без Zustand/Redux):

```
MapWorkspace (все useState)
  ├── TopToolbar ← activeTool, onToolChange
  ├── LeftPanel  ← objects[], selectedObjectId, onObjectSelect, onObjectDelete
  ├── MapCanvas  ← activeTool, objects[], onObjectCreate, onObjectUpdate, onObjectDelete
  │     ├── MapEvents → handleMapClick → switch(activeTool) → onObjectCreate(...)
  │     └── <Polyline>, <Marker icon={L.divIcon}> для рендера
  └── RightPanel ← selectedObject → MapObjectProperties → onSave(updatedObject)
```

**Цикл создания объекта:**
1. `TopToolbar.onToolChange('measure')` → `MapWorkspace.setActiveTool('measure')`
2. Клик на карте → `MapCanvas.handleMapClick` → при втором клике → `onObjectCreate({ type: 'measure', ... })`
3. `MapWorkspace.handleObjectCreate` → `setObjects([...objects, newObj])` → `setSelectedObjectId(newObj.id)` → `setActiveTool('select')`

### Как работает Leaflet-рендер

- Линии: `<Polyline positions={[latlng1, latlng2]} color={obj.color} />`
- Подписи: `<Marker position={midpoint} icon={segmentLengthIcon('123.45 м')} interactive={false} />`
- `segmentLengthIcon` уже реализован в `MapCanvas.tsx:602–623` — **переиспользуй**.
- Расстояние: функция `haversineDistanceMeters(p1, p2)` уже есть в `MapCanvas.tsx:1184` — **переиспользуй**.

### Правый клик на кнопке TopBar

`TopToolbar.tsx:510–513` — уже есть `onContextMenu` на кнопках инструментов, вызывает `onOpenCoordinateBuilder?.(tool.id)`.
Для «Измерить» нужна **другая** callback-пропса — `onMeasureClearAll?: () => void` — или обобщённый `onToolContextMenu(toolId)`.

Диалог подтверждения: переиспользуй существующий `AlertDialog` из shadcn/ui (уже есть в codebase, используется в других местах).
