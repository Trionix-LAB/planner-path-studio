# T-106: Отдельный цвет линий галсов в свойствах зоны

**Issue:** #106
**Spec:** New requirement

## Why
* Сейчас в свойствах зоны можно менять цвет зоны, но цвет линий галсов остается только глобальным.
* Оператору нужен независимый per-zone контроль цвета галсов рядом с настройкой цвета зоны.

## Goal
* Добавить в свойства зоны отдельный цвет галсов, который меняет именно цвет линий галсов этой зоны.

## Plan
* [x] Шаг 1: Обновить спецификации (`spec/spec.md`, `docs/screens.md`, `docs/mission-format.md`) с новым требованием и полем хранения.
* [x] Шаг 2: Расширить модель объекта зоны (`MapObject`) и mission-адаптеры для чтения/записи `lane_color`.
* [x] Шаг 3: Обновить рендер карты: линии галсов берут цвет из зоны (fallback на глобальный стиль).
* [x] Шаг 4: Обновить UI свойств зоны: рядом с цветом зоны добавить отдельный выбор цвета галсов.
* [x] Шаг 5: Обновить/добавить тесты и прогнать `npm run test`.

## Acceptance
* [x] В свойствах `survey_area` доступен отдельный контрол цвета галсов рядом с цветом зоны.
* [x] Изменение цвета галсов влияет только на линии галсов выбранной зоны.
* [x] Цвет галсов сохраняется в миссии и восстанавливается при повторном открытии.
